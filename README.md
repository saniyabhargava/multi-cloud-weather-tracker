# Multi-cloud-weather-tracker
A resilient weather tracking application deployed on AWS and Azure with automated DNS failover for disaster recovery. Provisioned using Terraform for infrastructure-as-code deployment.


## Technology Stack
- **Frontend:** React.js
- **Cloud Providers:** AWS (S3, Route 53), Azure (Blob Storage, Traffic Manager)
- **IaC:** Terraform
- **APIs:** OpenWeatherMap API

## Main Features
- Weather tracking with real-time updates
- Multi-cloud deployment for high availability
- Automated DNS failover during outages
- Fully automated provisioning with Terraform
- Global distribution for low-latency access
